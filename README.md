# ChatBot

## How to run?


1. Create a virtual environment

```bash
conda create -n ineuronChatBot python=3.10 -y
```


2. Activate the Environment

```bash
conda activate -n ineuronChatBot 
```

3. Install Packages

```bash
pip install -r requirements.txt
```

4. Chainlit Commands

```bash
chainlit init
```

```bash
chainlit run app.py
```
